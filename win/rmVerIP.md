## Descripción
Este comando muestra la configuración de red en sistemas Windows.

### Código
```batch
ipconfig
```

### Sinopsis
Muestra la configuración de red actual en Windows, incluyendo la dirección IP, máscara de subred y puerta de enlace predeterminada.
